# 🏆 FIFA World Cup 2026 — Road to the Final

An interactive bracket picker for the FIFA World Cup 2026.

**[→ Open the app](https://wc2026-road-to-the-final.pages.dev/)**

![screenshot](public/screenshot.png)

## Features

- Pick winners round by round from R32 through the Final
- Drag and drop to reorder teams within each group
- Undo / redo support
- Responsive — scales to fit any screen from tablet upward

## Development

```bash
npm install
npm run dev
```

## Tech Stack

- [Svelte 5](https://svelte.dev) + [Vite](https://vitejs.dev)
- [Tailwind CSS v4](https://tailwindcss.com)
- Deployed on [Cloudflare Pages](https://pages.cloudflare.com)
